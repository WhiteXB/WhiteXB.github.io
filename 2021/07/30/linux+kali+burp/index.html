linux+kali+burp

  • 2021-07-30
  • 作者 white-x
  • 685 字
本文最后编辑于 前，其中的内容可能需要更新。

linux下安装阿帕奇

  1. yum install httpd httpd-devel -y #安装

  2. systemctl start httpd.service #启动

  3. systemctl enable httpd.service #执行

  4. firewall-cmd –zone=public –add-port=80/tcp –permanent #打开80防火墙

  5. firewall-cmd –reload 确认

kali root 密码

  1. 开机e进入单用户模式

  2. 在Kali GNU GRUB页面，选择linux那一行，将ro 改为 rw 在此行末尾增加 init=/bin/bash 修改完毕以后按Ctrl+X 键进入单用户模式

  3. passwd命令重置root账号密码

kali xshell 连接

  1. 打开kali的终端 输入:leafpad /etc/ssh/sshd_config 打开配置文件

  2. 找到#PermitRootLogin prohibit-password

  3. 将后面的(prohibit-password )改成yes去掉#

  4. #passwordAuthentication 去掉#号

  5. 然后保存关闭

  6. 打开终端输入:service ssh start开启服务

  7. 然后再输入:service ssh status 确认开启状态

  8. 出现下面的提示就是服务开启成功了

kali burp suite 安装

1.jdk环境最重要 1.8

2.修改文件名可以做到快速替换

3.java -jar b【tap】执行破解软件

4.开始使用
